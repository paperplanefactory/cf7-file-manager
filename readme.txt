Questo plugin è stato sviluppato utilizzando Claude.ai

== Description ==

CF7 File Manager è un'estensione per Contact Form 7 che migliora la gestione dei file caricati tramite i form. Il plugin salva i file in modo sicuro e fornisce un'interfaccia amministrativa dedicata per visualizzare, scaricare ed eliminare i file.

Caratteristiche principali:
* Salvataggio sicuro dei file caricati
* Interfaccia amministrativa dedicata
* Gestione organizzata dei file per form
* Struttura cartelle per anno/mese
* Integrazione con Contact Form 7
* Supporto multi-file
* Protezione accesso ai file
* Integrazione con Flamingo
== Installation ==

1. Carica la cartella 'cf7-file-manager' nella directory '/wp-content/plugins/'
2. Attiva il plugin dalla pagina 'Plugin' in WordPress
3. Assicurati che Contact Form 7 sia installato e attivato
4. Vai su 'CF7 Files' nel menu amministrativo per gestire i file

== Frequently Asked Questions ==

= Dove vengono salvati i file? =
I file vengono salvati in una struttura organizzata all'interno della cartella uploads di WordPress, in /cf7-uploads/.

= Come posso proteggere i file caricati? =
Il plugin implementa automaticamente misure di sicurezza per proteggere i file caricati. Solo gli utenti autorizzati possono accedere ai file tramite l'interfaccia amministrativa.

= È compatibile con l'ultimo WordPress? =
Sì, il plugin viene regolarmente testato con le ultime versioni di WordPress e Contact Form 7.

= Come si integra con Flamingo? =
Il plugin aggiunge una riga nel box Meta di Flamingo con l'elenco dei file allegati e il relativo link.
