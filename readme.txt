Gestisce i file caricati tramite Contact Form 7 con un'interfaccia amministrativa dedicata.

== Description ==

CF7 File Manager fornisce un'interfaccia intuitiva per gestire i file caricati tramite Contact Form 7.
Caratteristiche principali:
* Organizzazione automatica dei file in sottocartelle per form
* Interfaccia amministrativa dedicata
* Filtri per tipo di file e ricerca
* Gestione sicura dei file
* Protezione delle cartelle di upload

== Installation ==

1. Carica la cartella 'cf7-file-manager' nella directory '/wp-content/plugins/'
2. Attiva il plugin dalla pagina 'Plugin' in WordPress
3. Vai in 'CF7 Files' nel menu di amministrazione
